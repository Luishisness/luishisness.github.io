MARGIN SECTION

Margins are used to add spacing between elements. Shorthand is margin:0px which would assign 0 to all top, right, left, and bottom margins. Shorthand can also be written as margin:0px 0px 0px 0px to specif targeting. Long hand margins seen in the below example can be defined as such margin-bottom:0xp and so forth.
